Smart Wardrobe App - Setup Instructions

FIRST TIME SETUP:
1. Make sure you have an internet connection
2. Double-click "setup.bat" and follow the instructions
3. This will download Node.js (if needed) and install the app

TO RUN THE APP:
1. Double-click "start-app.bat"
2. Wait for your browser to open automatically
3. The app will be available at http://localhost:3000

TO STOP THE APP:
- Close the black command window that opened
- Or press Ctrl+C in the command window
